## Near APIs

VM provides a convenient API to interact with the NEAR blockchain. There are three methods:
- [`Near.block`](#nearblock)
- [`Near.call`](#nearcall)
- [`Near.view`](#nearview)

## Near.block

 | param      |  required     | type               | description                                                           |
 |-----------|-----------|-------------------------|-----------------------------------------------------------------------|
 | `None`      |  **required** | object   | the path pattern(s)  |
